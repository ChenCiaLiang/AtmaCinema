# AtmaCinema ‚Äì Mobile App Cinema Booking System

AtmaCinema is a cinema ticket booking system consisting of:

- **Flutter Mobile App** (`Mobile`)
- **PHP/Laravel Backend** (`Laravel`)
- **Database Schema & SQL Dump** (`AtmaCinema.sql`)

This project is designed as a full-stack application where users can browse movies, view schedules, and manage bookings.
